FAQ

What is Slideshower?

Slideshower is the simplest app to run photo slideshows on macOS. It's designed for ease of use and functionality.

Is Slideshower easy to use?

Yes, Slideshower is user-friendly, allowing for simple, randomized, full-screen photo shows on macOS.

Can I shuffle photos in my slideshow?

Absolutely, Slideshower includes a shuffle mode to mix up your slideshow order, a feature not available in the macOS Photos app.

How much does Slideshower cost?

It's free to use up to 150 files. If you need to run a bigger slideshow, you can buy a one-time license for $19 only.

How do I download Slideshower?

You can download Slideshower directly from this website.

How to install Slideshower?

Download a dmg installation file and open it. Drag & drop Slideshower icon into the Application folder icon. And it's done! If you have problems, contact me at hello@slideshower.com

How to activate the license key?

If you bought the license, you should get your license key in the email. Copy it, open the app and click the button "Enter license key" located in the bottom left corner of the app. Paste your license key and activate it.

Which macOS versions are supported?

Slideshower runs on macOS Monterey (12.0) and up.

What file formats are supported?

Slideshower currently supports the following image formats: JPG/JPEG, PNG, HEIC, GIF, WebP, TIF, JXL and AVIF.

Can Slideshower handle folders with multiple subfolders?

Yes, Slideshower can import photos from folders containing multiple nested directories. When you select a folder, the app will automatically find and include all supported image files from the main folder and all of its subfolders, making it perfect for viewing huge photo collections organized in multiple directories.

Can Slideshower play background music while running a slideshow?

Starting with version 3.0, Slideshower supports playing background music during your slideshows. You can add multiple audio tracks that will play in sequence (in a loop or not). Supported audio formats are: .mp3, .wav, .aac, .aiff, .audio.

Can I request new features for Slideshower?

Yes! I am open to suggestions. If you need more features, reach out to me, and I'll consider adding them.
